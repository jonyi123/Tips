Tip Policy ‚Äî Plain English

  1. Daily-first. We process each calendar day separately, then sum the period.
  2. Base tips = Tips + Gratuity ‚àí Withholding (3% default, before splits).
  3. From each server‚Äôs base that day: BOH gets 30%, HB gets 10% (basis configurable), and the server keeps the rest. All are from the same base (not sequential), unless HB is set to ‚Äúof (base ‚àí BOH)‚Äù.
  4. Only staff who worked that day receive that day‚Äôs pools.
  5. BOH tiers split the day‚Äôs BOH pool: Lead 50% (Cook/Meat/Prep), Assist 35% (Cook/Meat/Prep Assist), Dishwasher 15%.
  6. Rounding: After every step (to cents).
  7. Credit card fee can be taken before (reduces base/pools) or after (reduces server keep only).
  8. For FOH/BOH-only: set HB % = 0 (use the preset).
